See `AGENTS.md` for the source-of-truth workflow for this template.

Fast rules:

1. Start with a preset if the user wants agency, consultancy, or studio positioning.
2. Edit `src/data/site.json`, `src/config/theme.json`, and `src/content/*` before changing components.
3. Do not hardcode one company identity into shared layouts.
4. Preserve static-first behavior, accessibility, SEO, and AEO structure.
5. Before finishing, run `npm run lint`, `npm run check`, and `npm run build`.
